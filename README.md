## 🔹 Kali Linux

```bash
sudo apt update && sudo apt upgrade
sudo apt install python3
sudo apt install git
```

Clone Repository:

```bash
git clone https://github.com/SahilAli9/python-iq-tester.git
```

Open Folder:

```bash
cd python-iq-tester
```

Run Project:

```bash
python3 quiz_project.py
```
